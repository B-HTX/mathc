## 4. 建模的关键假设

论文里最好先写假设，不然模型会散。

可以这样设：

### 假设1：机器人简化为多连杆系统

* 上肢攻击：可近似为“躯干 + 上臂 + 前臂”三连杆
* 下肢攻击：可近似为“躯干 + 大腿 + 小腿”三连杆
* 冲撞：可近似为整体质心平动碰撞模型

### 假设2：动作在一个固定时窗内完成

每个攻击动作有一个总时长 (T_i)，动作过程中角速度、角加速度可以用平均值或峰值表征。

### 假设3：有效攻击主要由末端速度和等效质量决定

对拳、腿类动作，攻击效果主要取决于：
[
E_i \sim \frac12 m_{eq,i} v_{i}^{2}
]
或冲量
[
J_i = m_{eq,i} v_i
]

### 假设4：自身稳定性由重心投影、支撑域和角动量扰动决定

动作越大，越可能导致质心投影逼近支撑边界，引起倾倒。

### 假设5：不考虑复杂柔性碰撞细节

接触时间、局部变形等采用等效参数近似。

这些假设是比赛型题目里非常常见且合理的。

## 5. 问题1的指标体系怎么设计

建议设计四类指标：

### A. 攻击效果指标

1. **末端线速度** (v_i)
2. **等效打击质量** (m_{eq,i})
3. **冲量** (J_i = m_{eq,i}v_i)
4. **动作耗时** (T_i)
5. **有效作用高度/命中关键区概率** (p_i)

综合后可以定义攻击收益：
[
A_i = \alpha_1 \hat J_i + \alpha_2 \hat p_i + \alpha_3 \(1-hat p_i)
]

其中 (\hat{}) 表示归一化。

---

### B. 稳定性指标

1. **质心水平偏移量** (\Delta x_i)
2. **角速度** (ω_i)
3. **单腿支撑比例** (s_i)

定义稳定风险：
[
R_i = \beta_1 \hat{\Delta x_i} + \beta_2 \hat ω_i + \beta_3 \hat s_i}
]

---

### C. 实修正项

1. 动作耗时 (T_i)
2. 连招可衔接性 (q_i)
3. 失手后暴露风险 (e_i)

定义：
[
Ui=γ1q^i−γ2e^i
]

其中：
qi：衔接性
ei：失手暴露风险

## 8. 双目标优化模型怎么建

到这一步，就可以正式建模了。

---

### 8.1 目标函数

对第 (i) 个动作，定义：

* 攻击收益 (A_i)
* 稳定风险 (R_i)
* 实战适用性 (U_i)

综合评分：
[
Scorei​=λAi​+(1−λ)Si​+εUi​
]

其中
[
Ai：攻击效果
Si：稳定性
Ui：小权重修正项
ε 取很小，比如 0.05～0.10
]

其中推荐：
λ=0.50
ε=0.10
也可以写：

* 主观赋权用AHP
* 客观赋权用熵权法
* 最终取组合权重

论文看起来会更完整。

## 9. 13种动作怎么做仿真与评分

做法很简单：

### 第一步：给每类动作设置参数

比如每个动作设9个变量：

* 末端速度 (v_i)
* 等效质量 (m_{eq,i})
* 命中关键区概率 (p_i)
* 动作时间 (T_i)

* 质心偏移 (\Delta x_i)
* 角速度 (ω_i)
* 单腿支撑比例 (s_i)

* 衔接性 (q_i)
* 失手暴露风险 (e_i)


这些值可以来自：

1. 机器人结构参数的理论估计
2. 文献经验值
3. 专家打分与仿真设定

数学建模比赛里，**“基于结构合理假设进行参数化仿真”是完全可以接受的**。

---

### 第二步：指标无量纲化

正向指标：
[
\hat x_i = \frac{x_i-\min x}{\max x-\min x}
]

逆向指标：
[
\hat x_i = \frac{\max x-x_i}{\max x-\min x}
]

---

### 第三步：求综合评分

代入
[
Score_i = w_1 A_i - w_2 R_i - w_3 C_i + w_4 U_i
]
