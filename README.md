## 4. 建模的关键假设

论文里最好先写假设，不然模型会散。

可以这样设：

### 假设1：机器人简化为多连杆系统

* 上肢攻击：可近似为“躯干 + 上臂 + 前臂”三连杆
* 下肢攻击：可近似为“躯干 + 大腿 + 小腿”三连杆
* 冲撞：可近似为整体质心平动碰撞模型

### 假设2：动作在一个固定时窗内完成

每个攻击动作有一个总时长 (T_i)，动作过程中角速度、角加速度可以用平均值或峰值表征。

### 假设3：有效攻击主要由末端速度和等效质量决定

对拳、腿类动作，攻击效果主要取决于：
[
E_i \sim \frac12 m_{eq,i} v_{i}^{2}
]
或冲量
[
J_i = m_{eq,i} v_i
]

### 假设4：自身稳定性由重心投影、支撑域和角动量扰动决定

动作越大，越可能导致质心投影逼近支撑边界，引起倾倒。

### 假设5：不考虑复杂柔性碰撞细节

接触时间、局部变形等采用等效参数近似。

这些假设是比赛型题目里非常常见且合理的。

## 5. 问题1的指标体系怎么设计

建议设计四类指标：

### A. 攻击效果指标

1. **末端线速度** (v_i)
2. **等效打击质量** (m_{eq,i})
3. **冲量** (J_i = m_{eq,i}v_i)
4. **动能** (E_i=\frac12 m_{eq,i}v_i^2)
5. **有效作用高度/命中关键区概率** (p_i)

综合后可以定义攻击收益：
[
A_i = \alpha_1 \hat J_i + \alpha_2 \hat E_i + \alpha_3 \hat p_i
]

其中 (\hat{}) 表示归一化。

---

### B. 稳定性指标

1. **质心水平偏移量** (\Delta x_i)
2. **零力矩点到支撑边界最小距离** (d_i)
3. **动作引起的角动量扰动** (L_i)
4. **单腿支撑比例** (s_i)
5. **恢复时间** (t_i^{rec})

定义稳定风险：
[
R_i = \beta_1 \hat{\Delta x_i} + \beta_2(1-\hat d_i) + \beta_3 \hat L_i + \beta_4 \hat t_i^{rec}
]

---

### C. 能耗指标

动作越复杂、幅度越大、持续越久，能耗越高。
可定义：
[
C_i = \int_0^{T_i}\sum_j \tau_j(t)\omega_j(t),dt
]

若没有详细轨迹，可近似成：
[
C_i \approx \sum_j \bar\tau_{ij}\bar\omega_{ij}T_i
]

---

### D. 实战适用性指标

1. 动作耗时 (T_i)
2. 连招可衔接性 (q_i)
3. 失手后暴露风险 (e_i)

定义：
[
U_i = \gamma_1(1-\hat T_i)+\gamma_2\hat q_i-\gamma_3\hat e_i
]


## 8. 双目标优化模型怎么建

到这一步，就可以正式建模了。

---

### 8.1 目标函数

对第 (i) 个动作，定义：

* 攻击收益 (A_i)
* 稳定风险 (R_i)
* 实战适用性 (U_i)

综合评分：
[
Scorei​=λAi​+(1−λ)Si​+εUi​
]

其中
[
Ai：攻击效果
Si：稳定性
Ui：小权重修正项
ε 取很小，比如 0.05～0.10
]

其中推荐：
λ=0.50
ε=0.10
也可以写：

* 主观赋权用AHP
* 客观赋权用熵权法
* 最终取组合权重

论文看起来会更完整。

## 9. 13种动作怎么做仿真与评分

做法很简单：

### 第一步：给每类动作设置参数

比如每个动作设：

* 动作时间 (T_i)
* 末端速度 (v_i)
* 等效质量 (m_{eq,i})
* 质心偏移 (\Delta x_i)
* 稳定裕度 (SM_i)
* 单腿支撑比例 (s_i)
* 能耗 (C_i)
* 命中关键区概率 (p_i)

这些值可以来自：

1. 机器人结构参数的理论估计
2. 文献经验值
3. 专家打分与仿真设定

数学建模比赛里，**“基于结构合理假设进行参数化仿真”是完全可以接受的**。

---

### 第二步：指标无量纲化

正向指标：
[
\hat x_i = \frac{x_i-\min x}{\max x-\min x}
]

逆向指标：
[
\hat x_i = \frac{\max x-x_i}{\max x-\min x}
]

---

### 第三步：求综合评分

代入
[
Score_i = w_1 A_i - w_2 R_i - w_3 C_i + w_4 U_i
]

---

### 第四步：排序与分类

把13种动作分成三档：

* **优先采用动作**
* **条件采用动作**
* **谨慎采用动作**

这比单纯排1到13更有实战意义。

## 10. 问题1最后的结论应该怎么写

在没有实际实验数据的情况下，从动力学和稳定性平衡出发，**通常比较合理的结论趋势**会是：

### 第一梯队：优先动作

1. **前踢**
2. **侧踢**
3. **直拳**
4. **膝撞**
5. **低扫腿**

原因大致是：

* 前踢：直线快、控制较好、攻击效率高
* 侧踢：爆发力强、距离合适时威胁大
* 直拳：节奏快、恢复快、稳定性好
* 膝撞：近身稳定、冲击集中
* 低扫腿：对破坏平衡很有效

---

### 第二梯队：条件采用

1. 组合拳
2. 拳腿组合
3. 冲撞
4. 勾拳
5. 摆拳

原因：

* 连招或中距离时很强，但对姿态、时机要求更高
* 冲撞收益大，但失手反噬明显
* 摆拳、勾拳更依赖横向躯干参与，稳定性不如直拳

---

### 第三梯队：谨慎采用

1. 回旋踢/转身踢
2. 五连踢
3. 倒地反击

原因：

* 这类动作理论输出可能高，但**动作时间长、失稳风险大、恢复成本高**
* 在机器人竞技里，稳定性通常比极限输出更关键，因为倒地会触发严重规则后果，如10秒内不能起身需人工复位，而每场人工复位最多2次 

所以问题1很适合得出一个结论：

> **团队优先选择“高攻击收益且低失稳风险”的中等复杂度动作，而非极限高爆发但高风险动作。**

这和题目要求是完全一致的。
