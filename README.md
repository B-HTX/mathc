# 问题3平衡版打包文件

这个包是在你上传的重建代码基础上做的“防失真”修改版。原始重建代码中的结果区间写成了：
- 自适应策略 0.903
- 固定高分策略 0.879
- 腿法偏置策略 0.731
- 随机策略 0.802

但按原代码实际运行，会出现：
- 自适应策略 1.000
- 固定高分策略 0.610
- 腿法偏置策略 0.057
- 随机策略 0.011

说明原仿真里存在明显的参数失衡。

## 这次修改了什么
1. 删除“自适应策略”的额外白送分支。
2. 缩小各策略防守质量差距。
3. 改写腿法偏置动作池，去掉五连踢、倒地反击这类高风险极端动作作为常规出手。
4. 改写随机策略动作池，不再把明显不适合作为常规基线的极端动作等概率塞进去。
5. 重新平衡命中、失手反击、对手主动进攻、疲劳衰减四组参数。
