# 问题2结果汇总

## 建模实现说明

- 采用 `攻击—防守匹配模型 + 防守闭环序列模型`。

- 攻击侧数据来自问题1结果表，并新增 `攻击原型 / 扰动风险 / 反制窗口 / 攻击强度`。

- 防守侧构造 22 个动作的能力向量 `(b,z,u,g,r,tau,k)`，再得到 `R_j` 与 `T_j`。

- 单动作评分采用 `总评分 = 0.5*P + 0.3*Q + 0.2*C`，再补出辅助动作与兜底动作，形成闭环。
