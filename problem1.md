## 4. 建模的关键假设

论文里最好先写假设，不然模型会散。

可以这样设：

### 假设1：机器人简化为多连杆系统

* 上肢攻击：可近似为“躯干 + 上臂 + 前臂”三连杆
* 下肢攻击：可近似为“躯干 + 大腿 + 小腿”三连杆
* 冲撞：可近似为整体质心平动碰撞模型

### 假设2：动作在一个固定时窗内完成

每个攻击动作有一个总时长 (T_i)，动作过程中角速度、角加速度可以用平均值或峰值表征。

### 假设3：有效攻击主要由末端速度和等效质量决定

对拳、腿类动作，攻击效果主要取决于：
[
E_i \sim \frac12 m_{eq,i} v_{i}^{2}
]
或冲量
[
J_i = m_{eq,i} v_i
]

### 假设4：自身稳定性由重心投影、支撑域和角动量扰动决定

动作越大，越可能导致质心投影逼近支撑边界，引起倾倒。

### 假设5：不考虑复杂柔性碰撞细节

接触时间、局部变形等采用等效参数近似。

这些假设是比赛型题目里非常常见且合理的。

## 5. 问题1的指标体系怎么设计

建议设计四类指标：

### A. 攻击效果指标

1. **末端线速度** (v_i)
2. **等效打击质量** (m_{eq,i})
3. **冲量** (J_i = m_{eq,i}v_i)
4. **动作耗时** (T_i)
5. **有效作用高度/命中关键区概率** (p_i)

综合后可以定义攻击收益：
[
A_i = \alpha_1 \hat J_i + \alpha_2 \hat p_i + \alpha_3 \(1-hat p_i)
]

其中 (\hat{}) 表示归一化。

---

### B. 稳定性指标

1. **质心水平偏移量** (\Delta x_i)
2. **角速度** (ω_i)
3. **单腿支撑比例** (s_i)

定义稳定风险：
[
R_i = \beta_1 \hat{\Delta x_i} + \beta_2 \hat ω_i + \beta_3 \hat s_i}
]

---

### C. 实修正项

1. 连招可衔接性 (q_i)
2. 失手后暴露风险 (e_i)

定义：
[
Ui=γ1q^i−γ2e^i
]

其中：
qi：衔接性
ei：失手暴露风险

## 8. 双目标优化模型怎么建

到这一步，就可以正式建模了。

### 8.1 目标函数

对第 (i) 个动作，定义：

* 攻击收益 (A_i)
* 稳定风险 (R_i)
* 实战适用性 (U_i)

综合评分：
[
Scorei​=λAi​+(1−λ)Si​+εUi​
]

其中
[
Ai：攻击效果
Si：稳定性
Ui：小权重修正项
ε 取很小，比如 0.05～0.10
]

其中推荐：
λ=0.50
ε=0.10

## 9. 13种动作怎么做仿真与评分

做法很简单：

### 第一步：给每类动作设置参数

比如每个动作设9个变量：第一点末端速度 (v_i)，第二点等效质量 (m_{eq,i}) ，第三点命中关键区概率 (p_i) ，第四点动作时间 (T_i)，第五点质心偏移 (\Delta x_i)，第六点角速度 (ω_i)，第七点单腿支撑比例 (s_i) ，第八点衔接性 (q_i)，第九点失手暴露风险 (e_i)

第一点， 末端速度 (v_i)
### 基本公式

对拳或腿末端，
[
v_i \approx k_i , L_i , \omega_i
]
其中：

* (L_i)：有效摆动半径
* (\omega_i)：动作角速度
* (k_i)：链式传动修正系数
---
### 你现在已知的结构参数
* 臂展 1.44 m，所以单臂有效长度大约可取 (0.60\sim0.72) m
* 小腿+大腿长度 686.5 mm，即腿长约 0.6865 m 
于是可以估计：
* **拳法有效长度**：(L_{arm}\approx0.65) m
* **腿法有效长度**：(L_{leg}\approx0.69) m

然后根据动作性质给一个角速度等级：
* 直拳：高
* 勾拳：中高
* 摆拳：中
* 前踢：中高
* 侧踢：高
* 回旋踢：很高，但准备时间长
* 五连踢：单次末端速度未必最高，但连续输出强





第二， 等效质量 (m_{eq,i})
打击时真正参与冲击的不是整机42 kg，而是：
* 摆动肢体本身的部分质量
* 躯干向末端传递的等效质量
所以等效质量可写成：
[
m_{eq,i} = \lambda_i M
]
其中 (M=42) kg 为整机质量，(\lambda_i) 为参与打击的质量比例。

### 推荐取值逻辑
根据动作类型给 (\lambda_i)：
* **拳法**：(\lambda_i = 0.06\sim0.12)
* **腿法**：(\lambda_i = 0.12\sim0.22)
* **冲撞**：(\lambda_i = 0.50\sim0.80)
* **倒地反击**：(\lambda_i = 0.10\sim0.18)
* 直拳：(m_{eq}\approx 42\times0.08=3.36) kg
* 前踢：(m_{eq}\approx 42\times0.16=6.72) kg
* 冲撞：(m_{eq}\approx 42\times0.65=27.3) kg
### 为什么这样合理
* 拳法主要是上肢和部分躯干参与；
* 腿法由更大质量链条参与；
* 冲撞本质是全身质心动量输出。
这个参数的目的不是还原真实碰撞，而是让不同动作在**冲量和动能指标**上能被比较。

第三点， 命中关键区概率 (p_i)
有效攻击需命中对方头部、躯干、四肢等关键区域，所以这个概率可以由以下因素决定：
[
p_i = f(r_i,; c_i,; q_i,; T_i^{-1})
]
其中：
* (r_i)：动作有效攻击距离匹配度
* (c_i)：轨迹控制精度
* (q_i)：对关键区覆盖能力
* (T_i^{-1})：越快越不容易被躲

### 怎么分级
可以按动作特征估：
* **高命中概率**：直拳、前踢、膝撞
  轨迹直接、控制精度高、目标明确
* **中等**：勾拳、摆拳、侧踢、低扫腿、冲撞
* **较低**：回旋踢、五连踢、拳腿组合
  动作复杂、路径长、容易被规避
* **特殊**：倒地反击
  依赖场景，不宜常规高估
### 一个很好用的定义
[
p_i = c_1 \hat{acc}_i + c_2 \hat{zone}_i
]
其中：
* (\hat{acc}_i)：动作控制精度评分
* (\hat{zone}_i)：对关键区命中覆盖评分



第四点， 动作时间 (T_i)
把动作分成三档：
短时动作：直拳、勾拳、膝撞Ti∈[0.25,0.45] s
中等动作：前踢、侧踢、摆拳、低扫腿、冲撞Ti∈[0.40,0.80] s
长时动作：回旋踢、五连踢、拳腿组合、倒地反击Ti∈[0.80,1.80] s
为什么这样分：
* 直拳、勾拳是短链快速输出；
* 腿法通常抬腿 + 伸腿 + 回收，时间更长；
* 回旋踢、五连踢涉及大角度转体和多阶段动作，必然更慢；
* 倒地反击还涉及起身过程，题目里防守动作写到“快速起身 1 秒内完成”，这可以作为时间量级参考 


第五点， 质心偏移 (\Delta x_i)
### 思路
动作过程中，摆动肢体会使整机质心横向或前后偏移。
用“摆动部分质量 × 位移 / 总质量”近似：
[
\Delta x_i \approx \frac{m_{mov,i}}{M} d_i
]
其中：
* (m_{mov,i})：动作中主要摆动部分质量
* (d_i)：该质量块相对机体中心的位移

### 如何理解
* 直拳：虽然手臂前伸，但整体偏移有限
* 摆拳：横向摆动更大，质心偏移更明显
* 前踢/侧踢：单腿离地，偏移显著
* 回旋踢：转体幅度大，偏移最大之一
* 冲撞：如果双足持续支撑，偏移不一定最大，但惯性扰动大
### 简化估计法
可直接按动作幅度分级：
* 小偏移：直拳、勾拳、膝撞
* 中偏移：摆拳、前踢、低扫腿、冲撞
* 大偏移：侧踢、回旋踢、五连踢、拳腿组合、倒地反击
再归一化：
[
\hat{\Delta x}_i = \frac{\Delta x_i-\min \Delta x}{\max \Delta x-\min \Delta x}
]




第六点， 角速度 (ω_i)
### 6.1 定义方式
设第 (i) 个动作的主导总转角为 ( \Theta_i )，动作总时长为 ( T_i )，则平均等效角速度定义为
[
\omega_i=\frac{\Theta_i}{T_i}.
]
### 6.2 主导总转角 ( \Theta_i ) 的构造
对不同动作，把真正贡献末端速度的转角加权求和：
[
\Theta_i=w_{\text{tr}} \theta_{\text{tr},i}+w_1\theta_{1,i}+w_2\theta_{2,i},
\qquad
w_{\text{tr}}+w_1+w_2=1.
]
其中：
* 上肢动作：( \theta_{\text{tr},i} ) 可取躯干/腰部带动角，( \theta_{1,i} ) 为肩部主摆角，( \theta_{2,i} ) 为肘部伸展角；
* 下肢动作：( \theta_{\text{tr},i} ) 为腰/髋转角，( \theta_{1,i} ) 为髋关节主摆角，( \theta_{2,i} ) 为膝关节伸展角；
* 冲撞：不以转动为主，可把 ( \omega_i ) 定义为“质心方向调整角速度”，或直接给很小值，仅作为稳定扰动修正项。
推荐权重：
* 拳法：( (w_{\text{tr}},w_1,w_2)=(0.20,0.50,0.30) )
* 腿法：( (0.25,0.50,0.25) )
* 回旋类：( (0.45,0.35,0.20) )
理由是：普通拳腿动作主要由肩/髋主导，回旋踢则腰部转体占比更高；而题目附录里确实把摆拳写成“腰部配合大弧度摆臂”，把回旋踢写成“腰部 320° 旋转带动腿部弧线扫踢”，所以把躯干转动显式纳入 ( \Theta_i ) 是合理的。
### 6.3 可直接给出的动作角度区间
你没法得到真实实验数据时，可以按动作结构分级给 ( \Theta_i )：
* 直拳：( \Theta_i \in [0.80,1.20]\ \text{rad} )
* 勾拳：( \Theta_i \in [1.00,1.50]\ \text{rad} )
* 摆拳：( \Theta_i \in [1.30,2.00]\ \text{rad} )
* 前踢：( \Theta_i \in [1.20,1.80]\ \text{rad} )
* 侧踢：( \Theta_i \in [1.40,2.10]\ \text{rad} )
* 低扫腿：( \Theta_i \in [1.50,2.30]\ \text{rad} )
* 膝撞：( \Theta_i \in [0.90,1.40]\ \text{rad} )
* 回旋踢：( \Theta_i \in [3.80,5.60]\ \text{rad} )
其中回旋踢的上界显著更大，是因为题目附录明确给了“腰部 320° 旋转”，即约 (5.59\ \text{rad})。同时，机器人基础参数中腰关节角运动空间也足够大，支持大幅转体建模。 


第七点， 单腿支撑比例 (s_i)
[
s_i = \frac{t_{single,i}}{T_i}
]
其中：
* (t_{single,i})：动作中单腿支撑时长
* (T_i)：动作总时长

### 动作特征判断
* 直拳、勾拳、摆拳、冲撞：通常双足支撑
  (s_i \approx 0\sim0.2)
* 前踢、侧踢、回旋踢、低扫腿、膝撞：明显单腿支撑
  (s_i \approx 0.5\sim0.95)
* 五连踢：单腿支撑比例很高
* 倒地反击：这个指标不适合直接用，可单独处理为“非稳定站立动作”


第八点， 衔接性 (q_i)
## 8. 衔接性 ( q_i )
衔接性本质上不是“这个动作强不强”，而是“这个动作打完后，能否自然接入下一动作或转入防守”。因此建议把 ( q_i ) 定义为**后续可行动作集合上的平均可达性评分**。
### 8.4 分档范围建议
可直接给区间：
* 高衔接：直拳、勾拳、组合拳、膝撞
  [
  q_i \in [0.70,0.95]
  ]
* 中衔接：摆拳、前踢、低扫腿、冲撞
  [
  q_i \in [0.45,0.75]
  ]
* 低衔接：侧踢、回旋踢、五连踢、倒地反击
  [
  q_i \in [0.15,0.50]
  ]
这和附录动作描述是一致的：组合拳、拳腿组合、五连踢本身就强调连击；但“五连踢”虽然链内衔接高，**对外衔接**未必高，因为整套动作结束后的回位慢、平衡负担大，所以在你这里最好把它定义为“链内输出强，但通用衔接性低于组合拳”。

第九点， 失手暴露风险 (e_i)
## 9. 失手暴露风险 ( e_i )
### 9.4 分档区间建议
* 低暴露：直拳、勾拳、膝撞
  [
  e_i \in [0.10,0.35]
  ]
* 中暴露：摆拳、前踢、低扫腿、冲撞、组合拳
  [
  e_i \in [0.30,0.60]
  ]
* 高暴露：侧踢、回旋踢、五连踢、拳腿组合、倒地反击
  [
  e_i \in [0.55,0.95]
  ]
尤其是倒地反击，题目和规则里都把“倒地”“快速起身”“倒地防御”单列出来，说明这本就是一个强场景依赖动作，不应按常规站立动作低估其暴露风险。
